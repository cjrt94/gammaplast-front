## Marca

Ver [`docs/BRAND.md`](docs/BRAND.md) para la paleta de color, tipografía y uso del logotipo. Fuente oficial: `brand/Manual-de-Marca-Gammaplast.pdf`.

- **Colores clave:** Verde Gamma `#00B140`, Negro `#000000`, Gris Claro `#A0A8AA`, Azul Gamma `#5056D3`, Lavanda Suave `#D2D4FF`
- **Tipografías:** Articulat CF (títulos/texto) + Corbel (párrafos)
